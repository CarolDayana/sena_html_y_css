Glosario de TI

A - M

A

Algoritmo: Conjunto de instrucciones definidas y ordenadas que permiten resolver un problema o realizar una tarea.

B

Backup: Copia de seguridad de datos que se utiliza para recuperarlos en caso de pérdida o daño.

C

Computadora: Dispositivo electrónico que procesa datos mediante programas para realizar tareas específicas.

D

Datos: Información procesada o sin procesar que puede ser utilizada por sistemas informáticos.

E

Encriptación: Proceso de codificar información para protegerla del acceso no autorizado.

F

Firewall: Sistema de seguridad que controla el tráfico de red entrante y saliente para bloquear accesos no autorizados.

G

Gigabyte: Unidad de medida de almacenamiento digital equivalente a aproximadamente mil millones de bytes.

H

Hardware: Conjunto de componentes físicos que forman parte de un sistema informático.

I

Internet: Red global de computadoras que permite la comunicación y el intercambio de información entre usuarios.

J

Java: Lenguaje de programación multiplataforma ampliamente utilizado en aplicaciones empresariales y móviles.

K

Kernel: Núcleo del sistema operativo que gestiona el hardware y permite que los programas se ejecuten.

L

Lenguaje de programación: Conjunto de reglas y símbolos que permiten escribir programas que puede interpretar una computadora.

M

Memoria RAM: Tipo de memoria volátil que almacena datos temporales mientras se ejecutan programas.
