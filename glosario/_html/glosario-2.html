N - Z

Orden de letras de la N hasta la Z

N

Navegador: Programa que permite acceder y visualizar sitios web a través de Internet.

O

Operador lógico: Símbolo usado en programación para combinar o modificar condiciones lógicas, como AND, OR y NOT.

P

Protocolo: Conjunto de reglas que determinan cómo se transmiten los datos en una red.

Q

Query: Consulta usada para extraer o manipular información en bases de datos.

R

Red: Conjunto de dispositivos interconectados que comparten recursos e información.

S

Servidor: Computadora o sistema que proporciona servicios o recursos a otras máquinas en una red.

T

Terminal: Interfaz de texto usada para interactuar directamente con el sistema operativo.

U

Usuario: Persona que utiliza software, hardware o servicios digitales.

V

Virtualización: Técnica que permite crear entornos simulados o virtuales dentro de un sistema físico.

W

Web: Conjunto de documentos y recursos interconectados a través de enlaces y accesibles mediante Internet.

X

Xamarin: Plataforma de desarrollo que permite crear aplicaciones móviles para Android e iOS usando C#.

Y

Yahoo: Uno de los primeros grandes portales web, que ofrecía servicios como correo electrónico y noticias.

Z

Zip: Formato de compresión de archivos que reduce su tamaño para facilitar el almacenamiento o transferencia.
